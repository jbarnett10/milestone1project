Origins of HTTP: 0.9 and 1.0

The web began at CERN (1989–1991) where Tim Berners‑Lee proposed a distributed hypertext system. The first stack paired URLs, HTML, and HTTP, demonstrated on info.cern.ch. The WorldWideWeb browser/editor showed HTTP could glue publication and retrieval.

HTTP/0.9 exposed a single GET returning an untyped payload. No headers; no content negotiation. HTTP/1.0 (1996) added request/response headers, media typing, and methods like HEAD and POST. Connections were typically short‑lived (one request per TCP connection), which later motivated performance work.

These early additions defined the request–response model and content typing that still shape the web. By introducing headers, methods, and media negotiation, HTTP/1.0 enabled richer pages and paved the way for later performance work.

Example: HTTP/1.0 Request

GET /index.html HTTP/1.0
Host: example.org

A simple HTTP/1.0 request with a Host header.
