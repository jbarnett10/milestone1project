About This Project

Author: Jayden Rodriguez-Barnett

Course: IT 3203 — Introduction to Web Development

Instructor: Denise Tucker

Project: Milestone 2 — Dynamic Frontend with Quiz

This site summarizes my research on the evolution of HTTP across versions 0.9/1.0, 1.1, 2, and 3, with emphasis on the 2022 specification refactor and QUIC's role in HTTP/3. The goal is to give students a clear, high-level overview of how HTTP has changed over time while preserving its core semantics.

For Milestone 2, I extended the original content with an interactive self-assessment quiz, additional explanations on key concepts, and small layout and style refinements. The quiz page allows visitors to test their understanding of HTTP versions, performance features, and protocol design trade-offs.

Technologies and Site Structure

This project is built with semantic HTML, a shared external CSS file, and a small amount of JavaScript for form handling and quiz logic. Navigation is consistent across all pages to make it easy to explore:

  • Home: Introduces the topic and motivation.
  • Origins: Early HTTP (0.9/1.0) and basic request/response ideas.
  • HTTP/1.1 & HTTP/2: Persistent connections, pipelining, and multiplexing.
  • HTTP/3 (QUIC): UDP-based transport, connection migration, and reduced latency.
  • Key Concepts: Core ideas that connect the different HTTP versions.
  • Quiz: Self-assessment on the evolution and design of HTTP.
  • References: Sources and further reading.
